# 寶可夢圖鑑

---

此為 Vue 做成的 SPA 頁面。

並串接 Pokemon 的 Restful API。

使用者可以藉由輸入寶可夢編號，取得寶可夢的樣子與基本資料。

請先安裝 modules 來使用。

```
npm install
```
